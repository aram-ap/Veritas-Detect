## How to Use

### ⚠️ Important: Enable File URL Access First!

To analyze local HTML files (file:// URLs), you must enable file access for the extension:

1. Go to `chrome://extensions/`
2. Find "Veritas: Misinformation Detector"
3. Click "Details"
4. Scroll down and enable "Allow access to file URLs" toggle

See **FILE_URL_SETUP.md** for detailed instructions with screenshots.

### Testing Steps

1. Open any HTML file in Chrome (drag & drop file into browser, or File → Open)
2. Click the Veritas extension icon
3. Click "Scan for Misinformation"
4. Take screenshots/recordings of the analysis

### Opening Demo Files

You can open these files by:
- Dragging the HTML file into your browser window
- Using File → Open File in your browser menu
- Typing the full file path in the address bar (e.g., `file:///Users/yourname/path/to/demo/high-trust.html`)
