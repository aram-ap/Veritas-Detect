## üö® Important Notes

### Domain Format
- Use root domains only: `harvard.edu` not `www.harvard.edu`
- Don't include protocol: `reuters.com` not `https://reuters.com`
- Don't include paths: `nytimes.com` not `nytimes.com/section/world`

### Wildcards (if supported)
Some search engines allow wildcards:
- `*.edu` (all educational institutions)
- `*.gov` (all government sites)
- `*.ac.uk` (all UK academic institutions)

Check if your Google Custom Search Engine supports this feature.

### Exclusions
Consider excluding:
- Social media platforms (Twitter, Facebook, Reddit)
- User-generated content sites (Medium, Quora)
- Low-quality news aggregators
